# SteamSuiteUtility

Native Windows helper for Steam Suite — idling, achievements, stats, and inventory commands.

Built from the sources in this folder. Output is copied to `libs/SteamSuiteUtility.exe` by `pnpm release` or the manual steps in [libs/README.md](../../libs/README.md).

Requires **.NET Framework 4.8** and the Desktop development with C++ workload when building Steamworks dependencies.
